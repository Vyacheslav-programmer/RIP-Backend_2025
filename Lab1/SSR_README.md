## Цель работы:
Разработка дизайна и базового интерфейса для системы оценки стоимости облачного хостинга

## Задачи:
- Создание дизайна 3 страниц в Figma
- Базовая шаблонизация в Django
- Интеграция с Minio для хранения изображений
- Реализация поиска и фильтрации услуг облачных провайдеров

## Страницы приложения:
1. **Главная страница** - список тарифов облачных провайдеров
2. **Страница тарифа** - детальная информация о выбранном тарифе  
3. **Страница заявки** - расчет стоимости выбранных тарифов

## Технологии:
- Django
- Figma
- Minio
- HTML/CSS/Bootstrap
- Docker
